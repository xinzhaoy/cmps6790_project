Introduction:

Interest in electric vehicles (EVs) is growing. To gain a comprehensive understanding of the electric vehicle landscape, I aimed to explore three key data sets using Washington state as an example: electric vehicle ownership data, population statistics, and EV charging stations.

Datasets

The following two datasets have all the necessary information I need in order to build prediction model and quantify discrimination in credit card application. I also provided descriptions of two datasets as well as how I am going to use them.

  • Dataset 1: Electric Vehicle Population Data

    This dataset shows the Battery Electric Vehicles (BEVs) and Plug-in Hybrid Electric Vehicles (PHEVs) that are currently registered through Washington State Department of Licensing (DOL).

  • Dataset 2: USA Census Income Data

    This dataset can give me Total income amount, Salaries and wages amount

  • Dataset 3: Alternative Fueling Stations from U.S. Department of Transportation

    This dataset can give me the number of EV station in the arae.

Question

    1: Are there discernible regional trends or hotspots for EV population?
    2: How is the level of income related to the likelihood of owning an electric vehicle?
    3: Consider your income as an employee, can we get an recommendation that should you buy a EV?

# Apply a logarithmic transformation to the data
log_income_data = np.log(income_data)
log_ev_count = np.log(ev_count)

# Define a function for the curve fit
def model_func(x, a, b, c):
  return a * np.exp(b * x) + c

# Fit the curve to the data with increased maxfev and adjusted initial guess
popt, pcov = curve_fit(model_func, log_income_data, log_ev_count, p0=[1, 0, 1], maxfev=5000)

# Generate fitted curve data
fitted_curve = model_func(log_income_data, *popt)

# Create the scatter plot for the transformed data
plt.figure(figsize=(10, 6))
plt.scatter(log_income_data, log_ev_count)
plt.plot(log_income_data, fitted_curve, "r-", label="Fitted Curve")
plt.xlabel("Log of Income")
plt.ylabel("Count of EVs (Log Scale)")
plt.title("Log of Count of EVs by Log of Income in each County")
plt.legend()
plt.show()

Now we can find following information:

Positive Correlation: The fitted curve is upward-sloping, indicating a positive correlation on the log-log scale. This means that as income increases, the number of EVs tends to increase as well.

Log-Log Transformation: Using a log scale for both the income and EV count can help linearize exponential relationships. This suggests that the relationship between income and EV count might be multiplicative rather than additive.

Model

  • Idea 1: EV Adpotion prodicet

    suggests a non-linear relationship between county income and EV count. This suggests that a model based on exponential growth might be suitable.

    Model equation:

    EV Count = a * exp(b * Income) + c where:

    EV Count: The number of electric vehicles in a county. Income: The average income in a county (logarithm transformed). a, b, c: Model parameters to be estimated. Parameter estimation:

    The parameters a, b, and c can be estimated using curve fitting techniques like the Levenberg-Marquardt algorithm. This involves minimizing the sum of squared errors between the predicted and actual EV counts.

    Model interpretation:

    The estimated parameters provide insights into the relationship between income and EV count.

    a: This represents the baseline EV count in a county with zero income. b: This represents the exponential growth rate of EV count with increasing income. A positive b value indicates that the EV count increases exponentially with income. c: This represents the error term in the model.
